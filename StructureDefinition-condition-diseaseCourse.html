Extension: DiseaseCourse

The clinical course of the disease (how the disease behaves over time), such as acute versus chronic.

The official URL for this extension is: 

https://www.napkon.de/fhir/StructureDefinition/condition-diseaseCourse

Status: active
Extension maintained by: Charité - Universitätsmedizin Berlin

Context of Use

This extension may be used on the following element(s):

  • Element

Guidance

This extension is used to indicate the disease course of a condition.

Usage info

Usage:

  • Use this Extension: Echocardiography Findings
  • Examples for this Extension: Condition/echo-finding-confirmed-icd-snomed, Condition/echo-finding-confirmed
